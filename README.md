# SIT725 – 2026 Trimester 1

Repository for **SIT725** (Deakin University), 2026 Trimester 1.

## Structure

- **1.3P/** – Practical 1.3 submission and related files
- Additional weekly subfolders will be added as the trimester progresses. Each subfolder typically contains a Node.js / Express.js task.

## Setup

Each task may have its own `package.json`. From a subfolder:

```bash
cd <subfolder>
npm install
npm start
```
